## 함수의 개념

### 개념 정리
* **함수(Function)** 는 반복되는 코드를 묶어 **재사용**할 수 있도록 만든 코드 블록  
* C언어에서 함수는 크게 세 부분으로 구성됨  
  1. **함수 선언 (Declaration)** : 컴파일러에게 함수의 존재를 미리 알림  
  2. **함수 정의 (Definition)** : 함수의 실제 동작 내용을 작성  
  3. **함수 호출 (Call)** : 작성한 함수를 사용하는 부분  

---

## 함수의 기본 구조

### 예시 코드
```c
#include <stdio.h>

// 함수 정의
void printWonyoungMind() {
    printf("오히려 좋아~~ 럭키비키잖아~~\n");
}

int main(void) {
    printf("코딩하다가 에러가 나도~~\n");
    printWonyoungMind(); // 함수 호출
    printf("원영적 사고 장착완료!\n");
    return 0;
}
````

### ✅ 포인트

* `void`는 **반환값이 없는 함수**를 의미함.
* 함수 호출 시 작성한 함수 이름을 그대로 사용.
* **`main()` 함수보다 아래에 작성하려면 선언(Prototype)** 이 필요함

## 매개변수와 반환값

### 개념 정리

* 매개변수(Parameter): 함수를 호출할 때 전달받는 **입력값의 변수명**
* 인자(Argument): 함수를 호출할 때 **실제로 전달하는 값**
* `return`은 **함수 실행을 종료하고 결과값을 호출한 쪽으로 돌려줌**

### 예시 코드

```c
#include <stdio.h>

int calculateTotalDamage(int weaponDamage, int skillDamage) {
    int totalDamage = weaponDamage + skillDamage;
    printf("데미지 계산중... 크리티컬!!\n");
    return totalDamage;
}

int main(void) {
    int myDamage = calculateTotalDamage(50, 35);
    printf("총 데미지: %d\n", myDamage);
    return 0;
}
```

### ✅ 포인트

* 함수 호출 시 넣는 값 → **인자(argument)**
* 함수 정의에서 받는 값 → **매개변수(parameter)**
* 반환값이 있다면 **`return` 키워드 필수**

## Call by Value (값에 의한 호출)

### 개념 정리

* 인자 전달 시, **값이 복사되어 함수로 전달**됨
* 즉, 함수 내부에서 값을 바꿔도 **원본 변수는 변경되지 않음**

### 예시 코드

```c
#include <stdio.h>

void failedLevelUp(int myLevel) {
    printf("> 특수 훈련소 입장! 현재 레벨: %d\n", myLevel);
    myLevel = 99;
    printf("> 훈련 완료! 현재 레벨: %d\n", myLevel);
}

int main(void) {
    int myLevel = 10;
    printf("훈련소 가기 전 레벨: %d\n", myLevel);

    failedLevelUp(myLevel);

    printf("훈련소 다녀온 후 레벨: %d\n", myLevel);
    return 0;
}
```

### ✅ 포인트

* `Call by Value`는 **원본이 아닌 사본을 전달**함.
* 따라서 함수 내에서 수정해도 호출자 쪽의 변수는 변하지 않음.
* 원본을 바꾸려면 **포인터(Call by Reference)** 를 사용해야 함

## 함수 선언과 정의

### 개념 정리

* **함수 선언(Function Declaration)** 은 컴파일러에게 함수의 형태를 알려주는 역할.
* `main()` 함수보다 아래에 정의하려면 반드시 선언이 필요함.
* 선언은 보통 헤더 파일(`.h`)에서 관리함.

### 예시 코드

```c
#include <stdio.h>

// 함수 선언
int add(int a, int b);
int subtract(int a, int b);
int multiply(int a, int b);
double divide(int a, int b);

int main(void) {
    int x, y;
    printf("두 개의 정수를 입력하세요: ");
    scanf("%d %d", &x, &y);

    printf("덧셈 결과: %d\n", add(x, y));
    printf("뺄셈 결과: %d\n", subtract(x, y));
    printf("곱셈 결과: %d\n", multiply(x, y));
    printf("나눗셈 결과: %.2f\n", divide(x, y));

    return 0;
}

// 함수 정의
int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }
int multiply(int a, int b) { return a * b; }
double divide(int a, int b) { return (double)a / b; }
```

### ✅ 포인트

* 선언부와 정의부는 **별개의 존재**이지만 반드시 **시그니처(매개변수형, 반환형)** 가 같아야 함.
* `double divide()`처럼 반환형이 실수라면 `(double)` 캐스팅이 필요함

## swap 함수 예제와 값 전달의 한계

### 예시 코드

```c
#include <stdio.h>

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main(void) {
    int x = 10, y = 20;
    swap(x, y);
    printf("x = %d, y = %d\n", x, y);
    return 0;
}
```

### 💬 출력 결과

```
x = 10, y = 20
```

### ✅ 포인트

* `swap()` 함수가 값만 복사해서 전달받기 때문에,
  **함수 내부에서 교환되어도 실제 변수에는 영향이 없음.**
* 이는 Call by Value의 한계이며, **Call by Reference(포인터)** 를 배우면 해결 가능

## 🎯 최종 정리

| 구분    | 설명                      | 예시                          |
| ----- | ----------------------- | --------------------------- |
| 함수 정의 | 실제 동작을 구현하는 부분          | `void printHello() { ... }` |
| 함수 선언 | 컴파일러에게 함수 형태를 알려줌       | `int add(int, int);`        |
| 함수 호출 | 정의된 함수를 사용하는 부분         | `add(3, 4);`                |
| 매개변수  | 호출 시 전달받는 값의 변수         | `int add(int a, int b)`     |
| 인자    | 실제로 전달하는 값              | `add(3, 4)`                 |
| 반환값   | 함수의 결과값                 | `return a + b;`             |
| 호출 방식 | Call by Value (값 복사 전달) | 원본 값은 변경되지 않음               |
