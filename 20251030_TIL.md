## 논리 연산자 (`&&`, `||`, `!`)

### 개념 정리

* `&&`: AND, 두 조건이 모두 참일 때만 참
* `||`: OR, 둘 중 하나라도 참이면 참
* `!`: NOT, 참과 거짓을 반대로 뒤집음

### 예시 코드

```c
int age = 20;
int height = 175;

if (age >= 19 && height >= 170)
    printf("입장 가능\n");
else
    printf("입장 불가\n");
```

### 포인트

* 괄호로 우선순위를 명시하면 오류를 예방할 수 있음
* 예: “성인이면서(AND) 키가 170 이상이면 입장 가능”

## 전위·후위 증감자 (`++i`, `i++`)

### 차이점

| 구분    | 동작 순서         | 설명                |
| ----- | ------------- | ----------------- |
| `++i` | 먼저 증가 → 사용    | 값을 1 증가시킨 후 사용    |
| `i++` | 먼저 사용 → 나중 증가 | 현재 값을 사용하고 나중에 증가 |

### 예시 코드

```c
int i = 3, j = 3;
int result1 = ++i; // i = 4, result1 = 4
int result2 = j++; // j = 4, result2 = 3
```

### 포인트

* 복잡한 식 안에 증감자가 섞이면 가독성이 떨어짐
* 증가는 별도의 라인으로 분리하는 것이 안전함

## 조건(삼항) 연산자 (`? :`)

### 구조

```c
조건식 ? 참일 때 : 거짓일 때
```

### 예시 코드

```c
int money = 9000;
printf("%s\n", money >= 10000 ? "플렉스!" : "학식");
```

### 포인트

* 단순한 조건 분기에 적합
* 복잡한 로직은 `if-else`로 처리하는 것이 가독성 좋음

## if / else if / else 조건문

### 개념

조건에 따라 서로 다른 코드를 실행함.
위에서부터 순차적으로 검사하며 처음 참인 조건만 실행됨.

### 예시 코드

```c
int score = 85;

if (score >= 90)
    printf("A등급\n");
else if (score >= 80)
    printf("B등급\n");
else if (score >= 70)
    printf("C등급\n");
else
    printf("D등급\n");
```

### 포인트

* 조건이 겹치지 않게 설계
* 큰 범위 → 작은 범위 순으로 정리하면 논리 흐름이 자연스러움

## switch-case 문

### 개념

하나의 변수에 대해 여러 경우를 간결하게 처리할 때 사용함.
정수형, 문자형 등 “값 기반 분기”에 적합함.

### 예시 코드

```c
int choice = 2;

switch (choice) {
    case 1:
        printf("커피 선택\n");
        break;
    case 2:
        printf("라떼 선택\n");
        break;
    case 3:
        printf("에이드 선택\n");
        break;
    default:
        printf("잘못된 선택입니다.\n");
        break;
}
```

### 포인트

* `break`를 생략하면 다음 case로 넘어감 (폴스루)
* 의도적인 폴스루는 주석으로 반드시 설명 남기기

---

## 실습 아이디어

1. **입장 판별기** — 나이, 키 입력 후 `&&`, `||`, `!` 조합 테스트
2. **증감자 비교기** — 전위/후위 차이 printf로 출력
3. **점심 추천기** — 금액에 따라 “플렉스” 또는 “학식” 출력
4. **성적 등급기** — 점수에 따라 등급 구분
5. **메뉴 선택기** — switch문으로 음료 메뉴 구현

---

## 🎯 최종 정리

* `&&`, `||`, `!` → 조건 결합의 기본
* `++i`, `i++` → 평가 순서 구분 명확히
* `? :` → 간단한 조건 처리에 적합
* `if-else` → 다양한 분기 처리용
* `switch-case` → 정수 기반 메뉴에 깔끔한 구조
