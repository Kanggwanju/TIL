### 2️⃣ C 프로그램의 구조
```c
#include <stdio.h>  // 표준 입출력 라이브러리

int main(void) {
    printf("Hello, C World!\n");
    return 0;
}
```

- `#include <stdio.h>` : 표준 입출력 기능(printf, scanf 등)을 사용하기 위한 선언  
- `int main(void)` : 프로그램이 시작되는 함수 (진입점)  
- `printf()` : 화면에 출력  
- `return 0;` : 프로그램 정상 종료 의미

---

### 3️⃣ 변수와 자료형

| 자료형 | 의미 | 크기 | 예시 |
|--------|------|------|------|
| `int` | 정수형 | 4바이트 | 1, -5, 100 |
| `float` | 실수형(소수점) | 4바이트 | 3.14 |
| `double` | 더 정밀한 실수 | 8바이트 | 3.141592 |
| `char` | 문자형 | 1바이트 | 'A', 'b' |

```c
int age = 25;
float height = 172.5;
char grade = 'A';

printf("나이: %d, 키: %.1f, 등급: %c\n", age, height, grade);
```

`printf` 서식 지정자  
- `%d`: 정수  
- `%f`: 실수  
- `%c`: 문자  
- `%s`: 문자열

---

### 4️⃣ 사용자 입력 받기
```c
#include <stdio.h>

int main(void) {
    int age;
    printf("나이를 입력하세요: ");
    scanf("%d", &age);  // & : 변수의 주소를 전달
    printf("당신의 나이는 %d살입니다.\n", age);
    return 0;
}
```

`scanf`는 입력받은 값을 메모리에 직접 저장하므로  
변수 이름 앞에 **& (주소 연산자)**를 붙여야 함

---

### 5️⃣ 컴파일과 실행 과정 (VSCode 기준)
1. 코드 작성 → `main.c`
2. 터미널에서 컴파일  
   ```bash
   gcc main.c -o main
   ```
3. 실행  
   ```bash
   ./main
   ```

참고: Windows에서는 `main.exe`, Linux/macOS에서는 `./main`
