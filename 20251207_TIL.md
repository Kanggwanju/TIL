## 1️⃣ 그래프(Graph)의 개념

### 💡 정의

* **그래프(Graph)** 는 **정점(Vertex)** 과 **간선(Edge)** 으로 이루어진 **비선형 자료구조**이다.
* 현실 세계의 다양한 관계를 표현할 수 있다.

  * 예: **지하철 노선도**, **SNS 친구 관계**, **지도 내 교차로 연결 구조**

---

## 2️⃣ 구성 요소

| 구성 요소                 | 설명             | 예시                |
| --------------------- | -------------- | ----------------- |
| **정점(Vertex / Node)** | 데이터를 담는 점      | 사람, 도시, 교차로       |
| **간선(Edge / Link)**   | 정점과 정점을 연결하는 선 | 도로, 지하철 노선, 친구 관계 |

---

## 3️⃣ 그래프의 종류

### 단순 그래프 (Simple Graph)

* 두 정점 사이에 **최대 하나의 간선만 존재**
* 자기 자신을 연결하는 **루프(Loop)** 없음

### 멀티 그래프 (Multi Graph)

* 두 정점 사이에 **여러 개의 간선이 허용**
* 예: 도시 간에 여러 도로가 존재할 수 있음

### 완전 그래프 (Complete Graph)

* 모든 정점이 **다른 모든 정점과 연결**
* 정점이 `n`개일 때 간선의 개수는 `n(n-1)/2`

---

## 4️⃣ 방향성에 따른 분류

| 구분                             | 설명                       | 예시           |
| ------------------------------ | ------------------------ | ------------ |
| **무방향 그래프 (Undirected Graph)** | 간선에 방향이 없음 → 양방향 이동 가능   | 친구 관계        |
| **방향 그래프 (Directed Graph)**    | 간선이 방향(화살표)을 가짐 → 단방향 이동 | 인스타그램 팔로우 관계 |

---

## 5️⃣ 차수(Degree)

| 용어                     | 설명                      |
| ---------------------- | ----------------------- |
| **진입 차수 (In-degree)**  | 특정 정점으로 **들어오는 간선**의 개수 |
| **진출 차수 (Out-degree)** | 특정 정점에서 **나가는 간선**의 개수  |

예시

* 노드 A → B, C

  * A의 진출 차수 = 2
  * B와 C의 진입 차수 = 1

---

## 6️⃣ 가중치 그래프 (Weighted Graph)

* 간선에 **추가 정보(가중치, Weight)** 가 포함된 그래프
* 가중치는 **거리, 비용, 시간** 등 다양한 의미로 사용됨

예시

* `서울 → 부산 (비용: 45,000원)`
* `A → B (거리: 10km)`

---

## 7️⃣ 그래프의 주요 개념

| 개념                | 설명                            |
| ----------------- | ----------------------------- |
| **인접(Adjacency)** | 두 정점이 **간선으로 직접 연결된 상태**      |
| **경로(Path)**      | 한 정점에서 다른 정점으로 가는 **간선들의 순서** |
| **사이클(Cycle)**    | 출발 정점과 도착 정점이 같은 **닫힌 경로**    |

---

## 8️⃣ 트리(Tree)와 그래프의 관계

* 트리는 **그래프의 한 종류**이다.
* 하지만, **트리는 사이클이 없고**, **모든 정점이 연결되어 있어야 한다.**

### ✅ 트리가 되기 위한 조건

1. **사이클이 없어야 함**
2. **모든 정점이 연결되어 있어야 함**

---

## 9️⃣ 트리(Tree) vs 그래프(Graph) 비교

| 구분       | 트리(Tree)     | 그래프(Graph)       |
| -------- | ------------ | ---------------- |
| 구조       | 계층 구조        | 네트워크 구조          |
| 루트 노드    | 반드시 존재       | 없음               |
| 사이클      | ❌ 없음         | ⭕ 존재 가능          |
| 부모-자식 관계 | 명확하게 정의됨     | 정의되지 않음          |
| 간선 수     | 정점 수 - 1     | 제한 없음            |
| 방향성      | 단방향 구조       | 방향 유무 선택 가능      |
| 대표 예시    | 조직도, 디렉토리 구조 | 지하철 노선도, SNS 관계망 |
