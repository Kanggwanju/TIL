## 1️⃣ 트리(Tree)의 개념

### 💡 정의

* **트리(Tree)** 는 **계층적(Hierarchical) 구조**를 가진 **비선형 자료구조**이다.
* 데이터 간의 **부모-자식 관계**를 표현하며, 하나의 루트(Root) 노드에서 여러 방향으로 뻗어 나간다.

트리는 현실 세계의 조직도, 폴더 구조, 게임 AI 탐색 등 다양한 곳에서 사용된다.

---

## 2️⃣ 트리의 구성 요소

| 용어                       | 설명                            |
| ------------------------ | ----------------------------- |
| **노드(Node)**             | 트리를 구성하는 기본 단위 (데이터를 저장)      |
| **루트(Root)**             | 트리의 시작점이 되는 노드                |
| **간선(Edge)**             | 부모와 자식 노드를 연결하는 선             |
| **부모 노드(Parent)**        | 자식 노드를 가지는 상위 노드              |
| **자식 노드(Child)**         | 부모 노드로부터 뻗어나온 하위 노드           |
| **형제 노드(Sibling)**       | 같은 부모를 가진 노드                  |
| **리프 노드(Leaf)**          | 자식이 없는 끝 노드                   |
| **내부 노드(Internal Node)** | 자식이 하나 이상 있는 노드               |
| **노드의 차수(Degree)**       | 한 노드가 가진 자식의 수                |
| **트리의 차수(Tree Degree)**  | 트리 전체에서 가장 큰 차수               |
| **레벨(Level)**            | 루트부터 몇 번째 층인지 나타내는 값          |
| **깊이(Depth)**            | 루트에서 특정 노드까지의 거리              |
| **높이(Height)**           | 노드에서 리프까지의 거리 (트리 높이 = 루트 높이) |

---

## 3️⃣ 이진 트리(Binary Tree)

### 💡 정의

* 모든 노드의 **차수(Degree)가 2 이하**인 트리
* 각 노드는 최대 **왼쪽 자식(left)** 과 **오른쪽 자식(right)** 을 가질 수 있다

---

## 4️⃣ 코드 구현

### 💻 구조체 정의

```c
typedef struct TreeNode {
    int data;
    struct TreeNode* left;   // 왼쪽 자식 노드
    struct TreeNode* right;  // 오른쪽 자식 노드
} TreeNode;
```

---

### 💻 노드 생성 및 연결

```c
TreeNode* n1 = (TreeNode*)malloc(sizeof(TreeNode));
TreeNode* n2 = (TreeNode*)malloc(sizeof(TreeNode));
TreeNode* n3 = (TreeNode*)malloc(sizeof(TreeNode));

n1->data = 10;
n2->data = 20;
n3->data = 30;

// n1을 루트로 설정
n1->left = n2;
n1->right = n3;

// 리프 노드는 NULL로 초기화
n2->left = NULL;
n2->right = NULL;
n3->left = NULL;
n3->right = NULL;
```

---

### 📘 메모리 해제

```c
free(n1);
free(n2);
free(n3);
```

> 동적 할당한 메모리는 사용 후 반드시 해제해야 함.
> 그렇지 않으면 **메모리 누수(memory leak)** 가 발생한다.

## 5️⃣ 시각적 구조

```
       (10)
      /    \
   (20)    (30)
```

* 10 → 루트 노드
* 20, 30 → 리프 노드
