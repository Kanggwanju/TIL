## 비트 연산자와 시프트 연산

### 개념 정리

* 비트(bit)는 0과 1로 구성된 데이터의 최소 단위이며, **1바이트는 8비트**로 구성됨.
* 정수는 이진수로 저장되며 **MSB(가장 왼쪽 비트)**가 부호를 나타냄.
* 음수는 ‘2의 보수’ 방식으로 표현함.

### 주요 연산자

| 연산자  | 의미                 | 예시             |
| ---- | ------------------ | -------------- |
| `~`  | NOT (비트 반전)        | `~13 → -14`    |
| `^`  | XOR (같으면 0, 다르면 1) | `13 ^ 10 → 7`  |
| `<<` | 왼쪽 시프트 (2배씩 커짐)    | `13 << 1 → 26` |
| `>>` | 오른쪽 시프트 (2배씩 작아짐)  | `13 >> 1 → 6`  |

### 예시 코드

```c
int a = 13, b = 10;
printf("~a: %d\n", ~a); 
printf("a ^ b: %d\n", a ^ b);

int num = 13;
printf("num << 1 = %d\n", num << 1);
printf("num >> 1 = %d\n", num >> 1);
```

### 포인트

* `<<`는 곱셈처럼, `>>`는 나눗셈처럼 작동하지만 **정수형 나눗셈이므로 소수점은 버려짐.**
* 비트 연산은 **속도가 빠르고 메모리 절약에 유리.**

## for문

### 개념

조건을 만족하는 동안 **정해진 횟수만큼 반복**할 때 사용.
초기식 → 조건식 → 실행문 → 증감식 순으로 수행됨.

### 예시 코드

```c
for (int i = 1; i <= 5; i++) {
    printf("%d번째 외침! 우리 르세라핌 폼 미쳤다.\n", i);
}
```

### 포인트

* 반복 횟수가 명확할 때 사용하기 적합.
* 변수의 초기값, 조건, 증감을 한 곳에서 관리할 수 있어 코드 가독성이 높음.

## while문

### 개념

**조건을 먼저 검사**한 후, 참일 동안 반복 실행함.
조건이 거짓이면 한 번도 실행되지 않을 수 있음.

### 예시 코드

```c
int i = 1;
while (i <= 5) {
    printf("%d번째 외침!\n", i);
    i++;
}
```

### 포인트

* 반복 횟수가 명확하지 않을 때 사용하기 좋음.
* 루프 조건이 실행 전에 평가됨.

## do-while문

### 개념

**조건을 나중에 검사**하기 때문에,
조건과 상관없이 최소 한 번은 실행되는 반복문.

### 예시 코드

```c
int choice;
do {
    printf("1. 새 게임 2. 불러오기 3. 종료\n메뉴 선택: ");
    scanf("%d", &choice);
} while (choice != 3);

printf("프로그램을 종료합니다.\n");
```

### 포인트

* 입력값을 한 번은 무조건 받아야 하는 상황에 적합.
* 메뉴 선택, 비밀번호 입력 같은 반복 구조에 자주 사용됨.

## 반복 제어문: break / continue

### 개념

* `break`: 반복문을 **즉시 종료.**
* `continue`: **현재 반복을 건너뛰고 다음 반복으로 이동.**

### 예시 코드

```c
for (int i = 1; i <= 10; i++) {
    if (i == 7) {
        printf("보물 발견! 탐색 종료!\n");
        break;
    } else if (i == 5) {
        printf("최고의 보물 발견!!\n");
        continue;
    }
    printf("%d번 상자를 엽니다.\n", i);
}
```

### 포인트

* `break`는 루프 전체를 종료할 때 사용.
* `continue`는 특정 조건에서만 실행을 건너뛸 때 사용.

## 중첩 반복문 (Nested Loop)

### 개념

반복문 안에 또 다른 반복문을 넣는 구조로,
**2차원 구조(예: 구구단, 별찍기)**를 표현할 때 사용됨.

### 예시 코드

```c
for (int level = 2; level <= 9; level++) {
    printf("===== 구구단 %d단 =====\n", level);
    for (int line = 1; line <= 9; line++) {
        printf("%d x %d = %d\n", level, line, level * line);
    }
}
```

### 포인트

* 바깥 for문은 단의 반복, 안쪽 for문은 곱셈 출력 담당.
* 반복이 중첩될수록 코드의 흐름을 시각적으로 구분하기 위해 **들여쓰기**가 중요.

## 🎯 최종 정리

* 비트 연산은 하드웨어 수준의 빠른 연산을 위해 사용되며, `~`, `^`, `<<`, `>>`가 핵심.
* 반복문(`for`, `while`, `do-while`)은 조건에 따라 명령을 여러 번 수행하는 구조.
* `break`와 `continue`로 반복 흐름을 제어하여 유연한 로직 구성 가능.
* 중첩 반복문을 활용하면 구구단, 별찍기 등 복합적인 패턴 구현이 가능.
